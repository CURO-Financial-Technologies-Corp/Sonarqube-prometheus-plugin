== Installation

1. Drop `sonar-prometheus-exporter-<VERSION>.jar` into `$SONARQUBE_HOME/extensions/plugins`.
2. Restart the SonarQube server.

[source,http]
----
GET /api/prometheus/metrics HTTP/1.1
